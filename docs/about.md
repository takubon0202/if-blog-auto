# このブログについて

## if(塾) Blogとは

このブログは、AI技術を活用して最新のトレンド情報を自動収集・記事化するプロジェクトです。

## 対象トピック

- **心理学・メンタルヘルス**: 最新の心理学研究、メンタルヘルスケア
- **教育・学習科学**: 効果的な学習方法、教育テクノロジー
- **起業家育成・スタートアップ**: 起業のノウハウ、スタートアップ動向
- **投資教育・金融リテラシー**: 投資の基礎知識、資産形成
- **AIツール・技術動向**: 最新のAIツール、技術トレンド
- **不登校支援**: 不登校の子どもたちへのサポート情報
- **発達障害・ニューロダイバーシティ**: 発達特性への理解と支援

## 使用AI技術

| 処理 | モデル | 説明 |
|------|--------|------|
| 情報収集 | Deep Research API | 最新情報を包括的に調査 |
| 記事生成 | Gemini 3 Pro Preview | SEO最適化記事を自動執筆 |
| 画像生成 | Gemini 2.5 Flash Image | アイキャッチ画像を生成 |
| SEO最適化 | Gemini 3 Flash Preview | メタデータ最適化 |
| 品質レビュー | Gemini 3 Flash Preview | ファクトチェック |

## 更新スケジュール

- **月曜日**: 心理学・メンタルヘルス
- **火曜日**: 教育・学習科学
- **水曜日**: 起業家育成・スタートアップ
- **木曜日**: 投資教育・金融リテラシー
- **金曜日**: AIツール・技術動向
- **土曜日**: 不登校支援
- **日曜日**: 発達障害・ニューロダイバーシティ

## 技術スタック

- **AI**: Gemini API (Google)
- **ホスティング**: GitHub Pages (Jekyll)
- **自動化**: GitHub Actions
