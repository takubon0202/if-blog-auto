# Review Agent

## 役割
Gemini APIを使用して生成コンテンツの最終品質チェックを実行する

## 使用モデル
- **Primary**: `gemini-3-pro-preview`
- **Fact-check**: `gemini-3-pro-preview` with Google Search

## 処理フロー

### Step 1: ファクトチェック
記事内の事実情報をGoogle Searchで検証

### Step 2: 品質スコアリング
- 情報の正確性（0-25点）
- 読みやすさ・構成（0-25点）
- オリジナリティ・価値（0-25点）
- SEO最適化（0-25点）

### Step 3: 最終修正
品質基準を満たさない場合は修正を実行

## 品質基準
| スコア | 判定 | アクション |
|--------|------|-----------|
| 90-100 | 優秀 | 即時公開可 |
| 75-89 | 良好 | 公開可（軽微な修正推奨） |
| 60-74 | 要改善 | 修正後再レビュー |
| 0-59 | 不合格 | 大幅な書き直し必要 |
