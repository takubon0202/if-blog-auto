# Storytelling Skill

## 概要
読者を惹きつけ、最後まで読ませるストーリーテリング技術を提供するスキル

## 目的
- 記事にストーリー性を持たせる
- 読者の感情に訴える
- 記憶に残るコンテンツを作成する

## 使用モデル
- `gemini-3-pro-preview`: ストーリー生成
- `gemini-2.0-flash`: クイック改善

## ストーリーテリングの5つの柱

### 1. フック（Hook）
読者の注意を最初の3秒で掴む

```
パターン:
- 衝撃的な統計: 「XX%の人が知らない事実があります」
- 問いかけ: 「あなたは〇〇について考えたことがありますか?」
- 対比: 「多くの人が〇〇だと思っています。しかし実際は...」
- パーソナルストーリー: 「〇〇さんがその日経験したこと」
- 危機感: 「今行動しなければ、あなたも...」
```

### 2. ヒーローズジャーニー（Hero's Journey）
読者が主人公として体験できる物語構造

```
構造:
1. 日常（Ordinary World）: 現状の課題
2. 冒険への呼びかけ（Call to Adventure）: 変化のきっかけ
3. 試練（Trials）: 困難と挑戦
4. 転換点（Transformation）: 気づきと成長
5. 帰還（Return）: 新しい日常と成果
```

### 3. Before → After 変換
変化を鮮明に描写

```
テンプレート:
【Before】
- 〇〇に悩んでいた
- △△ができなかった
- □□という状況だった

【After】
- 〇〇を克服した
- △△ができるようになった
- □□という理想の状態になった

【How（どうやって）】
- 具体的な3つのステップ
```

### 4. ケーススタディフォーマット
実在の事例を魅力的に伝える

```
構成:
1. 背景: 誰が、どんな状況だったか
2. 課題: 何に困っていたか
3. 決断: なぜ行動を起こしたか
4. 行動: 具体的に何をしたか
5. 結果: どうなったか（数値を含む）
6. 教訓: 読者が学べること
```

### 5. 感情カーブ
読者の感情を意図的に設計

```
感情の波:
1. 共感（低）: 読者の課題に寄り添う
2. 希望（上昇）: 解決の可能性を示す
3. 緊張（山場）: 課題の本質に迫る
4. 安心（解放）: 解決策を提示
5. 行動意欲（高）: 具体的なアクションへ
```

## ストーリー要素テンプレート集

### 導入ストーリー
```markdown
ある日、〇〇さん（XX歳・職業）は△△に気づきました。

「□□だと思い込んでいた自分が恥ずかしかった」と
当時を振り返ります。

しかし、この気づきが〇〇さんの人生を大きく変えることになります。
今では〇〇さんは△△を達成し、□□を実現しています。

この記事では、〇〇さんのような変化を実現するための
具体的な方法をお伝えします。
```

### 成功事例ストーリー
```markdown
### [人物名]さんのストーリー：[印象的なタイトル]

**状況**
〇〇さん（XX歳・職業）は、△△という課題を抱えていました。
毎日□□で悩み、「このままではダメだ」と感じていました。

**転機**
ある日、〇〇さんは△△という情報に出会いました。
「最初は半信半疑でしたが、試してみることにしました」

**行動**
〇〇さんが実践した3つのこと：
1. **〇〇を始めた**: 具体的な行動と期間
2. **△△を見直した**: 変更点と理由
3. **□□を習慣化した**: 継続のコツ

**結果**
3ヶ月後、〇〇さんの生活は大きく変わりました：
- △△が□□%改善
- 〇〇を達成
- 「今では△△ができるようになり、毎日が楽しい」

**あなたへのヒント**
〇〇さんの事例から、私たちが学べることは...
```

### 失敗から学ぶストーリー
```markdown
### よくある失敗パターン：[タイトル]

〇〇さんは△△を試みましたが、うまくいきませんでした。

**何が起きたか**
- 期待した結果: □□
- 実際の結果: △△
- ギャップの原因: 〇〇

**なぜ失敗したのか**
専門家の分析によると、この失敗には3つの要因がありました：
1. 〇〇を見落としていた
2. △△を過信していた
3. □□を軽視していた

**この失敗から学べること**
あなたが同じ轍を踏まないために：
- まず〇〇を確認する
- △△には注意する
- □□を優先する
```

### 驚きの発見ストーリー
```markdown
> **知っていましたか?**
>
> △△の最新調査（2026年1月発表）によると、
> 実は〇〇は□□であることがわかりました。
>
> これは多くの人が持っている「△△という常識」を
> 根本から覆す発見です。
>
> なぜこのような結果になったのでしょうか?
> 研究者たちは〇〇という要因を指摘しています。
```

## 感情を動かすフレーズ集

### 共感フレーズ
```
- 「あなたも〇〇で悩んでいませんか?」
- 「実は、多くの人が同じ課題を抱えています」
- 「〇〇が難しいと感じるのは、あなただけではありません」
- 「この記事を読んでいるあなたなら、きっと〇〇を経験したことがあるでしょう」
```

### 希望フレーズ
```
- 「しかし、解決方法があります」
- 「実は、〇〇を達成した人には共通点がありました」
- 「3ヶ月後、あなたも〇〇を実現できるかもしれません」
- 「今日から始められる第一歩があります」
```

### 行動促進フレーズ
```
- 「まずは〇〇から始めてみませんか?」
- 「今日できる最も簡単なアクションは〇〇です」
- 「この記事を読み終えたら、〇〇をしてみてください」
- 「あなたの最初の一歩を、今日踏み出しましょう」
```

## 品質チェックリスト
- [ ] 導入に強力なフックがあるか
- [ ] ストーリーにBefore → Afterの変化があるか
- [ ] 感情の波が意図的に設計されているか
- [ ] 具体的な人物や事例が含まれているか
- [ ] 読者が自分事として感じられるか
- [ ] 希望と行動意欲を与える終わり方か
- [ ] 押し付けがましくない表現か
